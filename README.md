# Scam Detector App 🛡️  

A Flask-based web application that detects scam-related keywords in text and leverages **OpenAI** to provide smarter scam awareness and explanations.  

## 🚀 Features  
- Detects scam keywords from user input
- Uses **OpenAI GPT API** to analyze suspicious text and explain why it might be a scam
- Simple web UI with Flask templates
- Provides scam awareness tips
- Lightweight and easy to run locally

## 🛠️ Tech Stack
- Python
- Flask
- HTML, CSS
- **OpenAI API**

## ▶️ How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/ScamDetectorApp.git
   cd ScamDetectorApp
   ```
2. **Install dependencies**
   ```bash    
   pip install -r requirements.txt
   ```
3. **Set up environment variables**
    ```ini  
   OPENAI_API_KEY=your_api_key_here
    ```
4. **Run the app**
   ```bash    
   python app.py
   ```
5. **Open in your browser**:  
   Go to [http://127.0.0.1:5000/](http://127.0.0.1:5000/) after starting the app.
